Bootstrap Grid Layout

La dimensione del testo di default di Bootstrap è di 16px.

Il font di default di Bootstrap è "Helvetica Neue", Arial, sans-serif.

Il margine superiore e inferiore di un paragrafo è di 1rem (16px).
